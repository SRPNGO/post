2 部件拆分原理

2.1 独体字

一般是部首汉字（如：金木水火土辶皿马皮日月目衣耳），全部看成部首，不能进一步拆分，仅由笔画构成。

自然码笔画码规则：

  1. 以横竖起笔的在 a 键："一丨亅レ乛フㄥ"
  2. 以点起笔的在 d 键（d=点）："丶冫氵"
  3. 以撇起笔的在 p 键（p=撇）："丿彡"

举例：金【jnp】（jn=金的自然码双拼，辅助码p=金的第一笔"撇"）

2.2 有明显部首的汉字

如：极、版、码、程、想、福、袋、鳌、游、洪、递

规则：辅助码 = 部首的声母（注：部首以《新华字典》为准，非按书写顺序）

举例1：架【jwm】（jw=架的双拼，辅助码m=部首"木"的声母）

举例2（双部首可选）：杏【xym】或【xyk】（部首可选"木"m 或"口"k，两者皆可）

2.3 不认识或非整体字部件的汉字

如：录、芈、暨、释、稽、躅、摭、谧、荔

规则：用"首笔画/末笔画"或"能认识的汉字"代替部首/部件

举例：躅＝足＋虫（可按"足"或"虫"的辅助码编码）

4 需要特殊记忆的部件（高频易混）
  • 日、月、曰、目 → 辅助码【o】（圆圆的o，对应圆形/椭圆形部件）
  • 扌（提手旁） → 辅助码【f】（f=扶手的"扶"）
  • 彳（双人旁） → 辅助码【x】（x=行人的"行"）
  • 一、丨、亅、乛（横竖折） → 辅助码【a】（基础笔画统一归a）
  • 亠（文字头） → 辅助码【d】（d=点起笔，文字头首笔为点）
  • 灬（四点底） → 辅助码【h】（h=火的变体，四点底是火的变形）
  • 艮 → 辅助码【g】（g=根的"根"，艮是"根"的声旁）
  • 肀（聿） → 辅助码【y】（y=聿的读音"yù"）
  • 耒 → 辅助码【l】（l=垒的"垒"，举例：耕【ggl】）
  • 爿 → 辅助码【p】（p=片的"片"，爿与片结构相似）
  • 豕 → 辅助码【u】（u=适的"适"，豕是"适"的部件）
  • 髟 → 辅助码【i】（i=长"cháng"的声母ch，对应i键）
  • 隹 → 辅助码【v】（v=锥"zhuī"的声母zh，对应v键）
  • リ（刀的变体） → 辅助码【d】（d=刀的"刀"，归为刀的变体）
  • （艹的变体） → 辅助码【c】（c=草的"草"，归为草字头变体）
  • 尢（尤的变体） → 辅助码【y】（y=尤的"尤"，结构相似）
  • 丌（齐的变体） → 辅助码【q】（q=齐的"齐"，结构相似）
  • 弋 → 辅助码【y】（y=弋的读音"yì"）
  • 厶 → 辅助码【s】（s=私"sī"的"私"，厶是"私"的部件）
  • ㄨ → 辅助码【x】或【w】（x=叉的"叉"，w=勿的"勿"，两者皆可）
